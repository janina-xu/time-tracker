# Daily Time Tracker

A full-stack personal productivity app to track daily sleep, work, study, and fitness time.

## Live Demo
[Coming soon]

## Features
- Log daily activities with start/end times
- Mark activities as "in progress" and update end time later
- View 7-day statistics with bar and pie charts
- Navigate between days to review past records
- Mobile-friendly design

## Tech Stack
- **Frontend**: Next.js, React, Tailwind CSS, Recharts
- **Backend**: Next.js API Routes
- **Database**: SQLite + Prisma ORM
- **Deployment**: Vercel (coming soon)

## Getting Started

```bash
# Install dependencies
npm install

# Set up database
npx prisma migrate dev

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
